Key Bank Client Challenge

Duration: Feb 2021 - May 2021 | Role: Data Analyst

Analyzed client transaction data for Key Bank to identify banking behavior trends among 18-24-year-olds. This project focused on understanding young adult banking patterns and developing innovative solutions to improve engagement with savings accounts and mobile features.

Key Responsibilities:

  • Queried and analyzed client transaction data using SQL
  • Identified banking behavior trends among 18-24-year-old demographic
  • Uncovered underutilization of savings accounts and mobile features
  • Formulated the "Smart Start Banking" program with gamified elements
  • Developed data-driven recommendations for client engagement

Technologies Used:

  • SQL for database querying and data analysis
  • Statistical analysis tools for trend identification
  • Data visualization software for presenting findings
  • Banking system databases and transaction records

Results:

  • Successfully identified key behavioral patterns in young adult banking
  • Discovered significant underutilization of savings accounts among target demographic
  • Uncovered low engagement with mobile banking features
  • Developed "Smart Start Banking" program with gamification elements
